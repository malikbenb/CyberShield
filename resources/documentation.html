Premiers pas avec CyberShield

Bienvenue dans la documentation CyberShield

CyberShield est une plateforme de pentesting innovante qui vous permet d'évaluer la sécurité de vos systèmes informatiques de manière autonome, sans nécessiter d'expertise technique approfondie.

Notre approche "Do Your Own Assessment" vous donne le contrôle total sur vos tests de sécurité, vous permettant de :

  • Identifier les vulnérabilités dans vos systèmes
  • Comprendre les risques associés
  • Obtenir des recommandations précises pour renforcer votre sécurité
  • Suivre vos progrès au fil du temps

Cette documentation vous guidera à travers toutes les étapes nécessaires pour tirer le meilleur parti de nos outils, de l'installation initiale à l'interprétation des résultats et à la mise en œuvre des correctifs recommandés.

Vue d'ensemble de la plateforme

La plateforme CyberShield se compose de plusieurs éléments clés :

Scanners

Nos scanners sont disponibles pour Windows, Linux et macOS. Ils analysent vos systèmes à la recherche de vulnérabilités connues, de configurations incorrectes et d'autres problèmes de sécurité.

Interface web

Notre interface web vous permet de gérer vos scans, de visualiser les résultats et de suivre vos progrès en matière de sécurité au fil du temps.

API

Notre API RESTful permet d'intégrer CyberShield à vos outils et workflows existants, facilitant l'automatisation des tests de sécurité.

Base de connaissances

Notre base de connaissances fournit des informations détaillées sur les vulnérabilités détectées et des conseils pour les corriger.

Installation

Prérequis système

Avant d'installer nos scanners, assurez-vous que votre système répond aux exigences minimales suivantes :

Scanner Windows

  • Windows 10 ou Windows Server 2016 (ou version ultérieure)
  • 2 Go de RAM minimum (4 Go recommandés)
  • 500 Mo d'espace disque disponible
  • Droits d'administrateur pour l'installation
  • .NET Framework 4.7.2 ou supérieur

Scanner Linux

  • Ubuntu 18.04+, Debian 10+, CentOS 7+, ou distribution équivalente
  • 2 Go de RAM minimum (4 Go recommandés)
  • 500 Mo d'espace disque disponible
  • Privilèges sudo pour l'installation
  • Python 3.6 ou supérieur

Scanner macOS

  • macOS 10.15 (Catalina) ou version ultérieure
  • 2 Go de RAM minimum (4 Go recommandés)
  • 500 Mo d'espace disque disponible
  • Droits d'administrateur pour l'installation

Installation sur Windows

  1. Téléchargez le scanner Windows depuis la section Scanners de notre site web.
  2. Double-cliquez sur le fichier d'installation téléchargé (CyberShield_Windows_Setup.exe).
  3. Si une alerte de sécurité Windows apparaît, cliquez sur "Plus d'informations" puis sur "Exécuter quand même".
  4. Suivez les instructions de l'assistant d'installation.
  5. Une fois l'installation terminée, lancez CyberShield depuis le menu Démarrer ou depuis le raccourci créé sur votre bureau.

Note : Pour la Version Pro, après l'installation, vous devrez activer votre licence en suivant les instructions fournies par email lors de votre achat.

Installation sur Linux

  1. Téléchargez le scanner Linux depuis la section Scanners de notre site web.
  2. Ouvrez un terminal et naviguez jusqu'au répertoire où vous avez téléchargé le fichier.
  3. Rendez le script d'installation exécutable :
chmod +x cybershield_linux_installer.sh
  1. Exécutez le script d'installation :
sudo ./cybershield_linux_installer.sh
  1. Suivez les instructions à l'écran pour terminer l'installation.
  2. Une fois l'installation terminée, vous pouvez lancer CyberShield en exécutant :
cybershield

Note : Pour la Version Pro, après l'installation, vous devrez activer votre licence en suivant les instructions fournies par email lors de votre achat.

Installation sur macOS

  1. Téléchargez le scanner macOS depuis la section Scanners de notre site web.
  2. Double-cliquez sur le fichier DMG téléchargé (CyberShield_macOS.dmg).
  3. Faites glisser l'application CyberShield dans votre dossier Applications.
  4. Lors du premier lancement, macOS peut vous avertir que l'application provient d'un développeur non identifié. Dans ce cas :
    • Ouvrez les Préférences Système > Sécurité et confidentialité.
    • Cliquez sur "Ouvrir quand même" pour autoriser l'application.
  5. Suivez les instructions à l'écran pour terminer la configuration.

Note : Pour la Version Pro, après l'installation, vous devrez activer votre licence en suivant les instructions fournies par email lors de votre achat.
